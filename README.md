##Oauth2是什么?
一种标准的用户认证授权协议


##Oauth2-server有哪些功能？

* 支持password,code,client等模式
    *  支持code自定义授权页面
    *  自持自定义校验
* springboot项目
* 采用shiro授权，cache使用redis,持久层使用jpa
